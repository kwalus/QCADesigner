Implemented Interfaces

QCADRadioButton implements AtkImplementorIface.

Description

GtkRadioButton widgets in versions of GTK <= 2.8.0 did not emit the "notify::active" signal when their active state was changed. A QCADRadioButton will emit such a signal by connecting to GtkRadioButton's "toggled" signal, and emitting its "notify::active" signal: 

static void instance_init (GObject *obj)
  {g_signal_connect (obj, "toggled", (GCallback)g_object_notify, "active") ;}

For versions of GTK > 2.8.0 detected at compile time, the definition of this widget evaluates to GtkRadioButton and none of the QCADRadioButton code gets compiled.
